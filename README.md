# 🧠 SYNAPSE

> **AI-Powered AIOps Platform for Automated Root Cause Analysis in Cloud-Native Microservices**

SYNAPSE is an intelligent AIOps platform that automates incident detection, root cause analysis (RCA), and AI-assisted troubleshooting for distributed microservice applications. It combines **Graph Neural Networks (GNNs)**, **Causal Inference**, **Retrieval-Augmented Generation (RAG)**, **Large Language Models (LLMs)**, and **Continual Learning** to identify failures and provide explainable solutions in real time.

---

## 🚀 Key Features

- 🔍 Automated Root Cause Analysis (RCA)
- 🧠 Graph Neural Network (GNN) based anomaly detection
- 🔗 Causal Graph Discovery & Validation
- 📚 Retrieval-Augmented Generation (RAG) with ChromaDB
- 🤖 AI-powered incident explanation using LLMs
- 📈 Real-time metrics dashboard
- 🌐 Interactive service dependency graph
- 🔄 Continual Learning for adaptive model improvement
- 📊 AI Evaluation & Benchmarking Framework
- ⚡ FastAPI backend with modular architecture
